# RDS(Amazon Relational Database Service의)
- `RDS`란 `AWS`에서 제공하는 분산 관계형 데이터베이스 서비스입니다.
- `AWS`에서 제공해주기 때문에 운영과 유지보수를 지원해주어 개발자는 코드 작성에 집중할 수 있습니다.
- `RDS`를 사용하는 것이 `EC2`에 `DB`를 다운 받아서 사용하는 것보다 비용이 더 세지만 애플리케이션을 더 빠르게 배포가 가능하고, 유지보수 측면에서 더 좋습니다.
## 특징
- 유연한 인스턴스 및 스토리지 확장
- 쉬운 백업과 복원
- 고가용성 확보
- `RDS` 암호화를 통한 보안성 강화
- DB Migration 서비스
## 지원하는 엔진
- **MariaDB**
- **Oracle**
- **MySQL**
- **Microsoft SQL Server**
- **PostgreSQL**
