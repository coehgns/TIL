# Components

- 리액트 컴포넌트는 입력을 받아서 정해진 출력을 내뱉음.
- 리액트 컴포넌트의 입력은 Props이고 출력은 리액트 엘리먼트임.
- 리액트 컴포넌트가 해주는 역할은 어떠한 속성들을 입력으로 받아서 그에 맞는 리액트 엘리먼트를 생성하여 반환해 줌.
