# 프로세스와 스레드
## 프로세스
- `프로세스`란 `OS`의 의해서 메모리 공간을 할당 받고, 실행되는 프로그램을 말합니다.
- 하나의 `프로세스`는 최소 한 개의 `스레드`가 존재하는데 그 `스레드`를 `메인 스레드`라고 합니다.

아래 이미지처럼 `프로세스`는 `OS` 내에서 각각 독립된 메모리 영역인 `CODE`, `DATA`, `STACK`, `HEAP을` 할당 받습니다. 이렇게 `프로세스`는 독립적이기 때문에 다른 `프로세스`에 접근할 수 없습니다.

`프로세스`가 다른 `프로세스`의 자원에 접근하기 위해서는 `IPC`를 사용해야합니다.

## 스레드
`스레드`는 `프로세스`의 작업 단위 혹은 실행 단위를 말합니다.

아래 이미지처럼 `프로세스` 내에서 각각의 `스레드`는 고유의 `STACK` 영역을 할당 받게 되고 `CODE`나 `DATA`, `HEAP` 영역은 공유하게 되는데 이러한 형태를 `멀티 스레드`라고 합니다. 이 때 한 `스레드`가 작업하여 자원을 변경한다면 `이웃 스레드`가 변경된 자원에 대해서 결과를 볼 수 있습니다.
