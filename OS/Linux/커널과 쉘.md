# 커널(Kernel)
- `커널`은 사용자와 `하드웨어` 사이에서 상호작용을 도와주는 OS의 핵심 구성 요소입니다.
- 하드웨어와 가장 가까이있으며, `쉘`로부터 내용을 전달받아 하드웨어를 조작합니다.
- 사용자가 직접 하드웨어를 조작하는 것이 불가능하고 하면 안되기 때문에 커널을 통해서 조작하게 됩니다.
# 쉘(Shell)
- 쉘은 사용자와 커널 사이에 위치하고 있으며, 사용자의 응용 프로그램과 커널이 통신할 수 있도록 도와줍니다.
- 사용자의 요청을 해석하여 커널에 넘기기 때문에 명령어 해석기라고도 불립니다.

다음 이미지처럼 쉘은 사용자가 명령어를 입력하면 해당 명령어를 해석 후 커널에 넘기게 되고, 커널은 해석된 명령어를 읽어 하드웨어를 제어하게 됩니다. 그 후 커널은 명령어 결과를 다시 쉘에 넘기고, 쉘은 사용자에게 결과를  응답하게 됩니다.
