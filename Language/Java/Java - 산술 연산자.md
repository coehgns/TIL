# 연산자

- `+`, `-`, `*`, `/`와 같이 계산을 수행하는 기호를 연산자라고 함.

## 연산자 종류

- 산술 연산자: `+`, `-`, `*`, `/`, `%`
- 증감 연산자: `++`, `--`
- 비교 연산자: `==`, `!=` , `>` , `<`, `>=`, `<=`
- 논리 연산자: `&&`, `||`, `!`
- 대입 연산자: `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- 삼항 연산자: `? :`

**연산자와 피연산자**

- 연산자(operator): 연산 기호
- 피연산자(operand): 연산 대상

**0으로 나누기**

- `10 / 0`과 같이 숫자는 0으로 나눌 수 없음. (수학에서 허용 X.)
- 0으로 나눌 시 프로그램에 오류가 발생함.
