- 객체지향 프로그래밍.
- Garbage Collector로 사용되지 않는 메모리를 자동적으로 정리해줌.
- 자바로 만들어진 프로그램은 JVM만 있으면 어디에서든지 실행 할 수 있음. JVM - 자바로 만들어진 프로그램을 실행해주는 프로그램.
