# DML(Data Manipulation Language)
- 정의된 데이터베이스에 입력된 레코드를 조회하거나 수정하거나 삭제하는 등의 역할을 하는 언어입니다.

## DML 종류와 역할
- select : 데이터 조회
- insert : 데이터 삽입
- delete : 데이터 삭제
- update : 데이터 수정

	데이터베이스 사용자가 응용 프로그램이나 질의어를 통하여 저장된 데이터를 실질적으로 처리하는데 사용하는 언어입니다.
	
	데이터베이스 사용자와 데이터베이스 관리 시스템 간의 인터페이스를 제공합니다.
