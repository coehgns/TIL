# **@ResponseBody**

- 자바 객체를 json 기반의 HTTP Body로 변환합니다.
- `@RestController`를 사용하는 경우 반환값에 자동으로 `@ResponseBody` 어노테이션이 붙게 됩니다.
- 자바 객체를 반환하면 알아서 Json 형식으로 매핑해서 응답합니다.
