# React - Rendering Elements

------

# Elements란?

- 리액트 앱을 구성하는 요소를 의미함.
- Elements는 화면에서 보이는 것들을 기술함.

------

## Elements의 생김새

- 리액트 Elements는 자바스크립트 객체 형태로 존재함.

```jsx
// 버튼을 나타내기 위한 엘리먼트

{
    type: 'button',
    props: {
        className: 'bg-green',
        children: {
            type: 'b',
            props: {
                children: 'Hello, element!'
            }
        }
    }
}
```
